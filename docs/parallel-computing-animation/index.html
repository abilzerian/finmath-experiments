Parallel Computing Animation

To run the animation checkout the project JavaGPUExperiments and launch com.christianfries.teaching.gpu.ParallelComputingAnimation via Maven exec.

Terminal / Console: 

				git clone https://github.com/cfries/JavaGPUExperiments
				cd JavaGPUExperiments
				mvn clean install exec:java -Dexec.mainClass=com.christianfries.teaching.gpu.ParallelComputingAnimation
